Flash card app, work in progress.


**** Main Menu ****

What would you like to do?
1. Create New Quiz     ****> Creates a new quiz!
2. Edit Existing Quiz  ****> Lets you add questions to existing quiz!
3. Start Quiz!         ****> Start taking an existing quiz!
4. Exit                ****> Leaves the program!

****

Quizzes are stored in the bin/quizzes folder in .txt format, so they can be
easily edited manually if you need. Currently the program can't delete questions.

Quiz name matches the file name.
